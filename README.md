#  Grossess – Application de Suivi de Grossesse

##  Description
**Grossess** est une application web dédiée au suivi de la grossesse.  
Elle permet d’accompagner les futures mamans avec des informations utiles, des conseils médicaux et un suivi personnalisé tout au long des différentes étapes de la grossesse.

Ce projet a été réalisé dans le cadre d’un projet académique en ingénierie informatique.

---

##  Fonctionnalités principales
-  Suivi des étapes de la grossesse (par semaine / mois)
-  Système de questions & réponses (type consultation)
-  Conseils et recommandations pour les mamans
-  Gestion des données utilisateurs
-  Authentification (login/register)
-  Interface moderne et responsive
